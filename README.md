# airflow_exchange_rates

Простая чекалка курса валютной пары RUB-USD, RUB-AMD, RUB-EUR

Опрашивает 1 раз в день, результаты записывает в postgres

#rate_report

забирает курсы за сегодняшний день и складывает их в csv
